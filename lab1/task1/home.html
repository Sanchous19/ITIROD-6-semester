О тренинге

JavaScript – один из самых популярных и кроссплатформенных языков программирования, позволяющий работать как с веб-интерфейсами так и с серверной частью и мобильными клиентами. Тренинг проводится под эгидой сообщества профессиональных фронтэнд-разработчиков Rolling Scopes. Тренинг посвящён современным подходам, технологиям и инструментальным средствам разработки веб-приложений с использованием языка программирования JavaScript.​

Пример кода

                function getFibonacciElement(index) {
                    let fibonacciRow = Array(index + 1);
                    fibonacciRow[0] = 0;
                    fibonacciRow[1] = 1;
                    for (let i = 2; i <= index; i++) {
                        fibonacciRow[i] = fibonacciRow[i - 1] + fibonacciRow[i - 2];
                    }
                    return fibonacciRow[index];
                }
